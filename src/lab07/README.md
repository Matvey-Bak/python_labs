## 3. Методы тестирования

### 3.1. Параметризованные тесты
Использован декоратор `@pytest.mark.parametrize` для множественных сценариев:
```python
@pytest.mark.parametrize("input_text,expected", [
    ("Привет, МИР!", "привет мир"),
    ("Hello World", "hello world"),
    ("", "")
])
```

### 3.2. Фикстуры pytest
Использована встроенная фикстура `tmp_path` для работы с временными файлами.

### 3.3. Интеграционное тестирование
Проверен полный цикл обработки данных:
- JSON → CSV → JSON
- Текст → токены → частоты → топ-N
